# Simple API 

HTTP Style as REST API

## Requirements
* Python
* Poetry
* Flask

## Install

```sh
poetry install
```

## Start dev server

```sh
poetry run flask --app rest_api:app run
```

## List

http://127.0.0.1:5000/cars

## By number

http://127.0.0.1:5000/cars/4


## Go to files for create, edit and delete

```sh
cd rest_api
```

## Create

```sh
python3 create.py
```

## Edit

```sh
python3 edit.py
```

## Delete

```sh
python3 delete.py
```
